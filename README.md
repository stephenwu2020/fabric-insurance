## Goal
- Let's figure out how hyperledger fabric and fabric-sdk-go work together in a web app.

## Tech
- hyperledger fabric v1.4.2
- fabric-sdk-go v1.4
- vue

## Prepare
- Before running this demo, you should able to run byfn v1.4.2, to make sure env is ok.

## Build
- make build

## Bring up network 
- make start-network

## Bring down network
- make stop-network

## Bring up server
- make start-server

## bring down server
- make stop-server

## test server api
- make test

## Experience
- localhost:8000
